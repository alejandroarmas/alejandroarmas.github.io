### What the heck is a .tar.gz?

To understand this, a GZ file is a file who has been compressed by the standard GNU zip algorithm used to encode information with fewer bits. The file being compressed has a ```.TAR``` extension, a type of file used to centralize multiple files into one called an archive. Together they form a ```.tar.gz``` file extension commonly referred to as a "tarball", which is very useful for sending multiple files at once or sharing them to your students in a class.

### A first function to define.

First we are going to define the member function for the GPS class so that we have a way to determine distance between two objects. 

According to the Haversine forumula, the way to determine the shortest distance over earth's surface barring hills is as follows:


$$ a = \sin(\frac{\triangle\phi}{2})^2 + \cos(\phi_{1}) \cdot \cos(\phi_{2}) \cdot \sin(\frac{\triangle\lambda}{2})^2$$

$$ c = 2 \cdot \arctan(\frac{\sqrt{a}}{\sqrt{1 - a}}) $$

$$ d = R \cdot c$$

Using the ```<cmath> (math.h)``` Library we are able to utilize a few trigonometric formulas and mathematical expressions useful for our calculation.

```c++
double GPS_DD::distance(GPS_DD& another)
{
  int earth_radius = 6371e3;
  double a, R, c, d, phi_1, phi_2, delta_phi, delta_lambda;

  phi_1 = deg2rad(this->getLatitude());
  phi_2 = deg2rad(another.getLatitude());

  delta_phi = deg2rad(another.getLatitude()-this->getLatitude());
  delta_lambda = deg2rad(another.getLongtitude()-this->getLongtitude());

  a = pow(sin(delta_phi / 2), 2) * cos(phi_1) * cos(phi_2) * pow(sin(delta_lambda / 2), 2);
  c = 2 * atan2(sqrt(a), sqrt(1 - a));
  d = R * c;

  return d;
}
```

For function members this logic is a bit different. We use it so that getter function does not make any changes to the member variable values.

```c++
std::string Person::getName() const
{
  return this->name;
}
```
This function returns a string of value = ```name```, specified by the ```std::string``` keyword by accessing the ```name``` variable in a read only mode. 

Similarly, it's useful in getter functions to use the ```const``` keyword in order to have your member function not create a copy of the argument. This is called having a constant function.
